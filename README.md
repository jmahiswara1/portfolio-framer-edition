# Gadang Jatu Mahiswara - Personal Portfolio Framer Edition

A modern, high-performance personal portfolio built with React, Vite, Tailwind CSS, and Framer Motion. This project features premium smooth scrolling via React Lenis and complex layout animations designed to showcase digital web development projects.

## Tech Stack

- React (v18)
- Vite
- Tailwind CSS (v3.4)
- Framer Motion (v11)

## Features

- Full-screen parallax hero section
- Smooth, inertia-based page scrolling
- Custom floating animated navigation menu
- Full-screen project cards with sticky stacking effects
- Dynamic statistics counter using framer-motion primitives
- Fully responsive design

## Getting Started

### Prerequisites

You need Node.js and npm installed on your machine to run this project.

### Installation

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally

To start the Vite development server:
```bash
npm run dev
```
Open `http://localhost:5173` to view it in the browser.

### Building for Production

To build the project for production deployment:
```bash
npm run build
```
The output will be generated in the `dist` folder.
